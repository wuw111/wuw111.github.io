# U-craftBE 项目信息

## 服务器信息
- 名称: U-craftBE Minecraft 服务器
- 类型: 基岩版 (Bedrock Edition)
- 运行时间: 2020年至今
- 版本支持: 1.21.X (生存服), 1.16-最新版 (小游戏服)

## 技术栈
- 前端: HTML5, CSS3, JavaScript
- 框架: jQuery
- 样式: 自定义CSS
- 图标: 自定义纹理

## 开发说明
1. 主样式文件位于 `css/main.css`
2. 主脚本文件位于 `js/main.js`
3. 所有页面共享相同的导航栏和页脚
4. 工具页面包含隐藏的彩蛋链接

## 维护指南
1. 更新服务器信息: 修改 `index.html`
2. 添加新工具: 在 `tools.html` 中添加新的工具卡片
3. 更新管理信息: 修改 `about.html`
4. 更改样式: 编辑 `css/main.css`
