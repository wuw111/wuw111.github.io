硬核级核心优势

动态数学公式计费

摒弃死板的固定收费。插件内置安全的 JS 函数沙盒，你可以针对 TPR、TPA、地标等配置 costFormula: "50 * Math.pow(1.05, count)"，让费用随玩家单日使用次数呈指数或线性增长，完美平衡服务器经济！

KVDatabase 玩家图鉴库

自动记录玩家的曾用名、所有使用过的 IP 地址、设备 ClientID 以及精准的累积在线时长。无惧熊孩子改名换姓，支持跨插件开放 API 直接读取玩家画像。

微秒级环形 TPS 监控

内存级环形数组处理 10 分钟满载数据 (12000 ticks)。摒弃传统的瞬时卡顿判断，为管理员提供近 30秒、2分钟、10分钟的真实 TPS 波动柱状图分析。

三位一体反作弊封禁

不仅支持本地基于 XUID、IP、设备的深度黑名单拦截，更无缝接入 BlackBE 与 UniteBan 云端反作弊接口，在玩家进服前即可实现云端拦截，并支持本地日志追溯。

全天候 CSV 安全审计

内置高性能日志队列，对全服所有经济流转、地标传送、查水表操作、后台资金修改等敏感操作进行 CSV 格式归档。有据可查，服主查账的好帮手。

完美的离线操作列队

对方不在线也能转账？使用计分板经济时，系统会将转账及管理员的罚款存入暂存列队，待目标玩家上线时自动结算。更有全局跨插件 API 支持你的其他扩展！
