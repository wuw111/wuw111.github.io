命令与权限列表

几乎所有指令都支持 GUI表单操作。当您不确定参数时，直接输入主命令即可呼出可视化菜单进行操作。

传送与地标

/tpr
普通玩家

执行随机传送。系统将自动为你寻找安全的着陆点（避开岩浆、虚空、墙体）。

/tpr

计费与冷却时间受动态公式影响。传送期间会附加抗性提升与失明效果以防卡顿伤害。

/tpa 及相关组合
普通玩家

玩家之间的互相传送请求系统。包括传送到对方身边，或将对方拉到自己身边。

/tpa
/tpacancel # 取消已发出的请求并按比例退款
/ac [玩家名] # 同意请求
/de [玩家名] # 拒绝请求
/tpaset # 开启/关闭接收 TPA 请求

支持财产限制（防止小号滥用）、支持手续费、支持过期自动退款。

/warp
普通玩家

前往服务器官方设定的公共地标。

/warp # 打开地标GUI列表
/warp go <地标名称>

家园系统

/home
普通玩家

管理个人家园，支持添加、删除、前往以及将家园“公开发布”。

/home # 打开家园主GUI
/home add <名称>
/home go <名称>
/home delete <名称>
/home publish # 打开发布公开家园GUI

随添加数量增多，添加家园的费用可通过数学公式成倍增长。

/homewarp
普通玩家

打开全服玩家发布的“公开家园”列表并进行传送。

/homewarp

基础生存

/back
普通玩家

传送回最近的死亡点。

/back # 默认传送到上一次死亡点
/back gui # 打开GUI选择历史死亡记录

支持返回越久远的死亡记录，可能产生的费用越高（可通过公式自定义）。

/suicide
普通玩家

立刻结束自己的生命。

/suicide
/tps
普通玩家

玩家查询当前服务器实时 TPS（需在配置文件开启 tps功能开关）。

/tps

经济与系统杂项

/moneysystem
普通玩家

呼出经济中心大厅，可向其他在线或离线玩家转账。

/moneysystem

支持税收。转账时可添加不超过 50 字的备注留言。支持向离线玩家打款缓存队列。

/languagesue
普通玩家

切换插件对您个人的提示语言（默认支持中英双语，视服主配置而定）。

/languagesue
/notice
普通玩家

重新查看服务器的欢迎公告。

/notice

管理员 - 玩家管理

/playermanage
OP 权限

全能玩家管理后台（支持视奸、替身说话、强制执行命令、增删资金、查阅背包NBT、崩溃客户端等）。

/playermanage # 打开在线玩家列表GUI
/playermanage <玩家名> <action> [参数]
# action 支持: look, talkas, cmdas, money, status, crash

此功能极为强大，其中背包 NBT 复制功能仅限 config 中的 SuperAdmin 使用。

/ban 与 /unban
OP 权限 控制台支持

三位一体封禁系统。支持记录封禁理由和时长（天）。

/ban # 打开封禁选择GUI
/ban <目标玩家名或XUID> # 控制台快捷封禁
/unban # 打开解封选择GUI
/unban <目标玩家名或XUID> # 控制台快捷解封

将同时记录玩家的名称、XUID、历史IP和设备ClientID进行多重拦截。

管理员 - 数据库与统计

/playerdatabase
OP 权限

底层 KVDatabase 玩家图鉴库管理。

/playerdatabase # 打开面板查看近期注册玩家概况
/playerdatabase refresh # 重算注册表数据
/playerdatabase export # 将 KVDB 全量导出为 JSON 备份
/playerdatabase query <名称片段> # 检索玩家曾用名及画像

检索功能可直接查阅已离线玩家的累计在线时长、历史IP，并支持对其进行离线封禁或资金扣除。

/tpsadmin
OP 权限 控制台支持

获取服务器 30s、2m、5m、10m 的平均高精度 TPS，并输出柱状图。

/tpsadmin

管理员 - 系统功能设置

/warpset 与 /homewarpadmin
OP 权限

地标管理及违规公开家园管理。

/warpset # 以当前坐标或指定坐标新增/编辑/删除公共地标
/homewarpadmin # 强制下架玩家发布的违规公开家园
/customcommand (或 /cuscmd)
OP 权限 (需SuperAdmin)

自定义命令映射系统。可在游戏内直接通过 GUI 创造新命令来映射原生指令或其它插件指令。

/customcommand

出于安全考虑，仅 config.json 中 customCommands.superAdmins 数组内的玩家可创建“以后台身份执行”的越权指令及执行删除操作。
