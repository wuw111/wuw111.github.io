Home Teleport Gold Calculator

Please enter a number from 0 to 150

Formula Description

This tool calculates the gold coins required for creating a new Home teleport point based on the server's formula:

Formula: Gold = 200 × Current Home Count + Current Home Count × 10^(1.2 + 0.005 × Current Home Count)

Results are rounded to the nearest integer.

Usage Tips

  • Home teleport points are an important feature that allows quick teleportation to set locations
  • Each new Home teleport point costs a certain amount of gold coins
  • The gold cost increases with the number of Homes you already have
  • Plan your Home locations wisely to avoid unnecessary gold waste
  • Calculation results are for reference only; actual costs are subject to in-game values
